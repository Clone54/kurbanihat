# QurbaniHat – Livestock Booking Platform

QurbaniHat is a modern livestock marketplace designed for the Qurbani season. It allows users to explore, filter, and book premium cows and goats from verified farms for their religious sacrifice.

## Live URL
[https://ais-dev-bxfrnywrkgpapwwq74djec-473656623930.asia-southeast1.run.app/](https://ais-dev-bxfrnywrkgpapwwq74djec-473656623930.asia-southeast1.run.app/)

## Purpose
The platform simplifies the process of finding healthy, Qurbani-compliant animals by providing a digital marketplace with detailed information, high-quality visuals, and a seamless booking experience.

## Key Features
- **Browse & Filter:** Explore a curated collection of cows and goats with sorting by price and category.
- **Detailed Insights:** View exhaustive details for each animal including breed, weight, age, location, and owner description.
- **Secure Authentication:** Firebase-powered authentication with Email/Password and Google Social Login.
- **Booking System:** Login-guarded booking form that allows users to register interest in specific animals.
- **User Dashboard:** "My Profile" section to view and update account information.
- **Modern UI:** Responsive design built with React, Tailwind CSS, and Framer Motion for a premium feel.
- **Informational Content:** Dedicated sections for Qurbani tips and top breed insights.

## NPM Packages Used
- `react` & `react-dom`: Core frontend framework.
- `react-router-dom`: SPA routing.
- `firebase`: Authentication and Cloud Firestore storage.
- `motion/react`: Smooth layout and element animations.
- `lucide-react`: High-quality iconography.
- `react-hot-toast`: Beautiful notification alerts.
- `lottie-react`: Interactive Lottie animations.
- `tailwindcss`: Utility-first styling.
- `clsx` & `tailwind-merge`: Utility for managing Tailwind classes.
